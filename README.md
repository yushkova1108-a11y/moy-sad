# Мой Сад · 21 день

PWA-приложение для челленджа самопознания.  
Выращивай своё дерево — каждое выполненное задание даёт новый рост.

---

## Как разместить на GitHub Pages

1. Создай новый репозиторий на GitHub (например `moy-sad`)
2. Загрузи в него три файла: `index.html`, `manifest.json`, `sw.js`
3. Перейди в **Settings → Pages**
4. В разделе **Branch** выбери `main` → `/root` → **Save**
5. Через минуту сайт будет доступен по адресу:  
   `https://ИМЯ_ПОЛЬЗОВАТЕЛЯ.github.io/moy-sad/`

---

## Как добавить на экран домой

### iPhone (Safari)
1. Открой ссылку в Safari
2. Нажми кнопку «Поделиться» (квадрат со стрелкой)
3. Выбери «На экран домой»
4. Нажми «Добавить»

### Android (Chrome)
1. Открой ссылку в Chrome
2. Нажми три точки → «Добавить на главный экран»
3. Нажми «Добавить»

---

## Как делиться деревом с подругой

1. Открой **Сад** → «Поделиться своим деревом»
2. Скопируй код и отправь подруге (в любом мессенджере)
3. Подруга открывает Сад → «+ Подруга» → вставляет код
4. Обновляй код раз в несколько дней — чтобы подруга видела актуальный прогресс

---

## Данные

Все заметки и прогресс хранятся **только в браузере** этого устройства.  
Никакие данные не передаются на серверы.
